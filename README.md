# 📚 Book Store System

This is a simple C# OOP-based console application that simulates a basic book store system with stock and shopping cart functionality.

## 🚀 Features
- Add books to the system
- Stock management system
- List available books
- Add books to cart
- Calculate total price in cart
- Automatically remove out-of-stock books
- Clear cart after purchase

## 🧠 Technologies Used
- C# (.NET Console Application)
- Object-Oriented Programming (OOP)
- Classes and Objects
- List<T>
- Loops and Conditionals
- ToString override

## 🎯 Purpose
This project was created to practice:
- OOP principles
- Class design
- List management
- Basic system simulation (store/cart logic)

## ⚙️ How to Run
1. Open the project in Visual Studio
2. Build the solution
3. Run Program.cs
4. Use the console menu
